# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/awaisbhatti267/portfolio.git
```

## Navigate To Project Folder

```bash
cd portfolio
```

## Install Dependencies

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
